# Ultra AI Intraday Stock Predictor

AI-powered stock prediction system for Indian stock market (NSE).

## Features

- RSI
- EMA
- MACD
- VWAP
- Bollinger Bands
- ATR
- ADX
- Stochastic RSI
- Supertrend
- Support/Resistance
- Multi Timeframe Analysis
- XGBoost AI Model
- Auto Learning System
- Dynamic Stoploss

## Technologies Used

- Python
- Pandas
- NumPy
- yfinance
- TA library
- XGBoost
- Scikit-learn
